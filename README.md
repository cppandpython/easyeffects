## рЯОѓ **What is EasyEffects?**

**EasyEffects** is a powerful audio processing application for Linux that provides real-time audio effects. These configuration files define the complete signal processing chain with:
- **Noise reduction** вАФ RNNoise neural network for voice noise suppression
- **Dynamic processing** вАФ Compressor, limiter, expander, gate
- **Frequency shaping** вАФ 15-band parametric equalizer
- **Spectral processing** вАФ De-esser, bass enhancer
- **Spatial effects** вАФ Reverb, crossfeed, stereo tools
- **Pitch shifting** вАФ Real-time pitch and tempo adjustment
